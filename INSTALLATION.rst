============
Installation
============

At the command line::

    $ pip install ynabintegrationscli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv ynabintegrationscli
    $ pip install ynabintegrationscli

Or, if you are using pipenv::

    $ pipenv install ynabintegrationscli

Or, if you are using pipx::

    $ pipx install ynabintegrationscli
